CRUDScope – HTML MediaPanel

Sistema de gestión de tareas con integración multimedia desarrollado en HTML semántico puro. Este proyecto demuestra el uso correcto de etiquetas semánticas, formularios, tablas y elementos multimedia.
Características

    Gestión de tareas con modelo CRUD (Crear, Leer, Actualizar, Eliminar)

    Formulario de creación con campos para título, descripción, prioridad, fecha y estado

    Listado de tareas en formato tabla con acciones de actualización y eliminación

    Sección multimedia con soporte para imágenes, audio y video

    Estructura semántica que mejora la accesibilidad y SEO

    Navegación interna mediante anclas

Secciones principales
Crear tarea

Formulario completo para registrar nuevas tareas con los siguientes campos:

    Título de la tarea (requerido)

    Descripción detallada

    Prioridad (escala 1-5)

    Fecha de creación

    Estado (Pendiente, En progreso, Completada)

Listado de tareas

Tabla que muestra todas las tareas registradas con:

    Título y estado actual

    Fecha de creación

    Botones de acción (Actualizar/Eliminar)

Multimedia

Galería de contenido multimedia que incluye:

    Imagen de ejemplo del panel

    Guía en formato audio

    Video demostrativo del sistema

Acerca de

Información sobre el propósito educativo del proyecto y el uso de HTML semántico.
Tecnologías utilizadas

    HTML5 con etiquetas semánticas (header, nav, main, section, article, aside, footer)

    CSS externo (styles.css)

    Elementos multimedia nativos (img, audio, video)

    Formularios HTML con validación básica

Instalación

    Clona o descarga el proyecto

    Asegúrate de tener la carpeta imagevideomusik/ con los archivos multimedia

    Abre index.html en tu navegador

bash
# No requiere instalación de dependencias
# Simplemente abre el archivo HTML

Uso

Navega por las diferentes secciones usando el menú superior:

    Crear: Accede al formulario de registro de tareas

    Listado: Visualiza las tareas existentes

    Multimedia: Explora el contenido multimedia del sistema

    Acerca de: Conoce más sobre el proyecto

Objetivos educativos

Este proyecto sirve para:

    Practicar el uso de etiquetas semánticas vs no semánticas

    Entender la estructura jerárquica de encabezados (h1-h6)

    Implementar formularios accesibles

    Integrar elementos multimedia en HTML

    Mejorar la accesibilidad web y SEO
